Detailed Description

3D math engine.

Author:
Stephane Dallongeville
Date:
08/2011

This unit provides 3D transformation methods :

  • translation X, Y, Z
  • rotation X, Y, Z
  • one directionnal light
  • 2D projection

    Can transform (including 2D projection) about ~10000 vertices / seconde.
