SGDK is a small, open and free development kit for the Sega Megadrive.

It contains a development library (sources included) which offer facilities to make software on Sega Megadrive / Genesis system.
You can find doxygen documentation about it in the 'doc' directory : doc/html/index.html

It also contains GCC compiler tools but for Windows system only.

Official Website: http://code.google.com/p/sgdk/

Don't forget to check tutorials on this page: http://code.google.com/p/sgdk/w/list
